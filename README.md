# Apollo Reborn Custom Source

This repository hosts the public AltStore feed and release assets for a custom Apollo Reborn build.

- Source URL: `https://raw.githubusercontent.com/nhaneezy/apollo-reborn-dist-b601f358/main/apps.json`
- Bundle ID: `com.nhanpham.ApolloReborn`
- Variant: `Liquid Glass`

The IPA assets in Releases are rebuilt from upstream Apollo Reborn releases and retargeted for this custom bundle identifier.
